# Certificate Images

Place your certificate images in this directory with the following naming convention:

- `meta-frontend.png` - Meta Front-End Developer Certificate
- `advanced-react.png` - Advanced React Certificate
- `html-css.png` - HTML/CSS in Depth Certificate
- `javascript.png` - JavaScript Programming Certificate
- `ui-ux.png` - UI/UX Design Principles Certificate
- `coding-interview.png` - Coding Interview Preparation Certificate
- `data-science.png` - Data Science Fundamentals Certificate
- `fullstack.png` - Full-Stack Development Certificate

**Note:** If images are not available, the portfolio will display a fallback placeholder automatically.
